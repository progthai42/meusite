Bem-vindo ao Curitour! Este site foi criado para ser o seu guia completo para explorar a cidade de Curitiba, capital do Paraná. Se você é um visitante curioso ou um morador local em busca de novos lugares para conhecer, o Curitour reúne informações sobre os melhores pontos turísticos, atividades culturais, gastronomia e muito mais. Curitiba é uma cidade repleta de história, cultura e inovação, e nosso objetivo é apresentar a você tudo o que torna esse lugar único. Ao navegar pelo Curitour, você encontrará sugestões de passeios que vão desde os tradicionais pontos turísticos até opções mais alternativas e secretas da cidade. Cada recomendação foi cuidadosamente escolhida para oferecer a melhor experiência possível aos nossos usuários. Seja para um dia de turismo ou para descobrir o que fazer nos fins de semana, o Curitour é o seu companheiro ideal para aproveitar Curitiba ao máximo. Explore as opções e prepare-se para viver momentos inesquecíveis na capital paranaense!

Tratamento de Elementos

Os elementos nesta página foram criados usando:

  • HTML: Foi utilizado para fazer o esqueleto das páginas e elementos padrões de navegação. Assim como, linkar os scripts de arquivos javascript.
  • CSS: Foi utilizado um arquivo único para estilizar as páginas, utilizando diversos seletores e combinadores.
  • JavaScript: Foram criados 3 arquivos para a páginas: Index.html, formAction.html e about.html. Os scripts foram usados para concatenar e criar dinamicamento elemento html na DOM. Assim como, foram usados para guardar e ler coockies de status de cadastro, nome e email de usuarios, afim de possibilitar o fluxo de uso do usuário desejado e esperado.
